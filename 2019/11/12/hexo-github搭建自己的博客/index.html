    hexo+github 搭建自己的博客

    组件
    创建时间:2019-11-12 10:58
    阅读:
    1. 环境配置
    2. 创建仓库
    3. 创建项目
    4. 更换主题(推荐)
    5. 上传项目
    6. 访问博客
    7. 补充问题

    环境配置

    • github账号
    • node.js, npm安装
    • git安装

    创建仓库

    • 在github上Repositories上新建一个仓库,仓库的命名必须为 你的github名字.github.io
    • 配置ssh key(这个自行百度吧,只要用过github,大概都会吧)

    创建项目

    • 创建一个空目录,目录名字即为你的项目名字
    • 进入目录,右键 Git Bash Here
    • 安装 npm install -g hexo , 初始化 hexo init , 启动hexo s(访问http://localhost:4000可看到内容)

    更换主题(推荐)

    • 主题地址
    • 选择自己喜欢的主题,拷贝地址
    • 下载主题 git clone https://github.com/yelog/hexo-theme-3-hexo.git
    • 修改主题 _config.yml 中 theme: landscape 修改为 hexo-theme-3-hexo
    • 先清理下pulic内容 hexo clean 启动hexo s 查看新主题

    上传项目

    • 修改配置 _config.yml 中
    • 安装发布插件 npm install hexo-deployer-git --save
    • 发布 hexo d

    访问博客

    • http:yourGitHub.github.io
